BigInt.cpp, BigInt.h, BigIntException.h - непосредственно реалзиация библиотеки
Testing.h - пространство имён, включающая в себя необходимые для тестирования функции
_get_cpu_time.cpp - позаимствованная мной кроссплатформенная функция для вычисления процессорного времени
main.cpp - основной файл приложения
Tests/test_gen.py - генератор тестов на Python. Сами тесты не включены из-за их огромных размеров
Tests/correctness_res.csv - результаты теста на корректность выполнения операций
Tests/time_*.csv - результаты тестов скорости для основных операций:
    io - ввод/ввывод
    ba - операции +, -, *, /, %
    ua - унарный минус
    comp - сравнение
    shifts - сдвиги влево и вправо
Tests/graph_*.png - соответствующие графики. На оси иксов - число разрядов чисел во внутреннем представлении
Tests/makeplot.gp - скрипт gnuplot, строящий графики по csv-файлам
